# stgsharp.vss.dev

A VSS/Foundation development repository in the [StgSharp ecosystem](https://github.com/Nitload-NSI/StgSharp).

This repository is intended to host the early foundation work for VSS-oriented abstractions, with current focus on:

- state foundations
- service foundations
- presenter contracts

At this stage, the repository does **not** aim to provide a complete graphics backend or a full GUI/layout system.

## Included projects

- `StgSharp.Common`
- `StgSharp.Native`
- `StgSharp.UserInterface`

`StgSharp.Native` is currently retained as a required low-level dependency layer, including native support code that the current StgSharp-based runtime still relies on.

This repository follows the StgSharp naming and licensing conventions and is intended to remain compatible with future upstream convergence.
